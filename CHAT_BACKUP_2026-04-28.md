- Отдельного API сообщений раньше не было.

## Что реализовали
1. Добавили персистентное хранение сообщений задач в SQLite:
   - таблица `task_messages` в `lib/db.ts`.
2. Добавили API сообщений задачи:
   - `GET /api/tasks/[id]/messages`
   - `POST /api/tasks/[id]/messages`
   - файл: `app/api/tasks/[id]/messages/route.ts`.
3. Обновили UI чата в `components/TaskBoard.tsx`:
   - загрузка истории из API при открытии задачи,
   - отправка сообщений через API,
   - отображение сохраненных сообщений.
